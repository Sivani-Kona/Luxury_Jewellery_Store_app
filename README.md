# 💎 Luxury Jewellery Store Web Application

Developed a responsive and user-friendly Luxury Jewellery Store web application using the Django framework to provide an elegant and seamless online shopping experience for jewellery products. The application was designed with both frontend and backend integration to efficiently manage products, users, shopping carts, and orders.

The project focuses on building a functional e-commerce platform with secure authentication, dynamic product management, and a visually attractive luxury-themed interface. It demonstrates full-stack web development skills, including frontend design, backend development, database handling, and user management.

## ✨ Key Features

* User Registration and Login Authentication
* Secure User Account Management
* Product Listings with Images and Descriptions
* Product Detail Pages for Individual Jewellery Items
* Shopping Cart Functionality
* Add to Cart and Remove from Cart Features
* Order Placement and Order History Tracking
* Responsive Design for Mobile and Desktop Devices
* Elegant Luxury Jewellery UI Design
* Admin Panel for Managing Products and Orders
* Category-wise Product Display
* Dynamic Database Integration using Django ORM
